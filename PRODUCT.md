# Product Specification — Nguyen Thanh Tam Portfolio

## Audience
- **Primary**: DevOps/Cloud Engineering Hiring Managers, Technical Recruiters, Infrastructure Team Leads.
- **Secondary**: Fellow developers and DevOps practitioners seeking cooperation or project reviews.

## Brand Positioning
- **The Transition Story**: From IT Support/Helpdesk to Cloud Engineering. Demonstrating a strong foundation in hands-on IT operations, systems administration, and support, coupled with modern cloud native engineering practices (Docker, CI/CD, Observability, System Design).
- **Core Voice**: Technical, precise, practical, and solution-oriented. Avoid flowery marketing fluff; describe systems by their actual stack and metrics (e.g., ticket count, database sizes, scrape intervals).

## Core Value Proposition
- Shows a progressive learning path mapping practical IT support experience to cloud-native systems.
- Demonstrates production-first thinking: secure secrets management, comprehensive liveness/health checks, built-in monitoring, and clean documentation.
